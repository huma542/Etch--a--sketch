# Etch-a-Sketch

A browser-based Etch-a-Sketch built as part of [The Odin Project](https://www.theodinproject.com/)
Live demo:huma542/Etch--a--sketch.git

Draw, hover, and erase on a dynamic grid. Think of it as the digital version of the classic toy.

## ✨ Features
- **Dynamic Grid**: Choose any grid size from 16x16 to 100x100
- **Hover to Draw**: Move your mouse over squares to "draw"
- **Color Modes**:
    - **Black**: Classic mode
    - **RGB/Random**: Each square gets a random color
    - **Progressive Darkening**: Colors get 10% darker each time you hover
- **Controls**: 
    - Clear/reset button
    - Grid size slider/input
    - Toggle grid lines on/off
- **Responsive**: Works on desktop and tablet

## 🛠️ Tech Stack
- **HTML5** - Grid structure
- **CSS3** - Styling, Flexbox/Grid layout
- **Vanilla JavaScript** - DOM manipulation, event listeners

No libraries or frameworks used. Pure JS.
